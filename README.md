# FitTrack - Фитнес Трекер

PWA-приложение для комплексного отслеживания фитнеса: тренировки, питание, БАДы и замеры тела.

## Возможности

- **Тренировки**: Записывайте упражнения, подходы, повторения и вес. Таймер отдыха между подходами.
- **Питание**: Дневник питания с подсчётом КБЖУ, трекер воды, интеграция с Open Food Facts.
- **БАДы**: Управление добавками с напоминаниями о приёме через push-уведомления.
- **Замеры тела**: Отслеживание веса и объёмов (грудь, талия, бёдра, шея и др.) с графиками прогресса.

## Технологии

- React 18 + TypeScript
- Vite + PWA
- Tailwind CSS
- Dexie.js (IndexedDB)
- Zustand (state management)
- React Router

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## PWA

Приложение работает как Progressive Web App:
- Устанавливается на телефон/десктоп
- Работает офлайн
- Push-уведомления для напоминаний

## Безопасность

- Все секреты хранятся в `.env` (не коммитится)
- Шаблон переменных окружения в `.env.example`
- Данные хранятся локально в IndexedDB
